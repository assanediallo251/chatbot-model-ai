# Questions de Test pour le Chatbot ISI

Ces questions peuvent etre posees apres avoir indexe le PDF de demonstration dans l'application.

## Questions simples

1. Qu'est-ce que l'Institut Superieur d'Informatique ?
2. Quelle est la mission principale de l'ISI ?
3. Quels sont les domaines de formation cites dans le document ?
4. Quelle est l'approche pedagogique de l'ISI ?
5. Pourquoi les projets sont-ils importants dans la formation ?

## Questions sur les competences

6. Quelles competences un etudiant doit-il developper a l'ISI ?
7. Pourquoi Git est-il mentionne dans le document ?
8. Quelles competences ne se limitent pas au code ?
9. Quel est le role de la documentation technique ?
10. Pourquoi la communication est-elle importante dans un projet informatique ?

## Questions sur le chatbot IA

11. Quel est l'objectif du chatbot ISI ?
12. Comment fonctionne le projet de chatbot IA ?
13. Que signifie l'approche RAG ?
14. Pourquoi utilise-t-on PostgreSQL avec PgVector ?
15. Quel est le role des embeddings dans le projet ?
16. Quel est le role du modele LLM dans le chatbot ?
17. Que doit faire le chatbot si l'information n'est pas dans les documents ?

## Questions sur les technologies

18. Quelles technologies sont citees dans le document ?
19. Pourquoi FastAPI peut-il etre utile pour ce projet ?
20. A quoi sert Streamlit dans l'application ?
21. Quel est le role de Docker ?
22. Pourquoi utiliser des bibliotheques de traitement PDF ?

## Questions plus longues

23. Explique comment le chatbot ISI peut aider les nouveaux etudiants.
24. Resume les objectifs pedagogiques de l'ISI.
25. Donne un resume du document en cinq phrases.
26. Explique le lien entre les projets pedagogiques et le monde professionnel.
27. Presente les etapes principales du projet chatbot IA.

## Questions pieges pour verifier la fiabilite

28. Quelle est l'adresse exacte de l'ISI ?
29. Quel est le montant des frais d'inscription ?
30. Qui est le directeur actuel de l'ISI ?

Pour ces trois dernieres questions, le chatbot devrait repondre que l'information n'est pas disponible dans le document fourni.
